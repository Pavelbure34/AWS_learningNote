AWS technical Essentials

AWS Services

Amazon Web Services is a web service that allows easy maintenance and use of cloud computing resources.
    AWS main customers are
  • Enterprises for its effectiveness, expendablitiy, low initial investment.
  • govt. organizations for its security and credence
  • startups for its low initial investment
    Advantages
  • No need for pre-configuration for expansion.
  • Easy distribution.
  • No investment for maintenance for data center
  • fast and effective services
  • access to latest hardware
  • Flexibility for configuration

On-premise vs AWS

  • Security
    Firewall, ACL, Admin ----> Security Group, Netowork ACL, and AWS lam
  • Networking
    Router, network pipeline, Switch ----> Elastic Load Balancing, and Amazon VPC.
  • Server
    on-premise server ----> AMI to Amazon EC2 instances.
  • Storage and database
    SAN, DAS, NAS ----> EBS(==SAN),EFS(==DAS),S3,RDS(==SQL)

AWS Global Infrastructure

  • Region: geographic location which consist of at least two areas.
  • Area: a cluster for data center.
  • Edge Location: designed to provide fast services locally and closest.
