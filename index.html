Security in Cloud

Security Group

Virtual Firewall that controls the traffic(IP address, Port Number).
  • It creates group of instances under the same firewall.
  • State Pool inspection:when certain traffic is in, that traffic is allowed to leave.
  • Used by Tier group such as Web Server, App Server, ...
  • Chain Diagram:For the source,you can either IP address, Security group!

NACL(Network Access Control List)

firewall in the subnet level.
  • in and outbounds are allowed by default.
  • it needs specific conditions of in and outbound traffic.
  • USing both NACL and Security Group ensures safety.
